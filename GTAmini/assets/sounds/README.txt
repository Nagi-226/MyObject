# 音效文件说明

请在此目录放置以下音效文件：

## 武器音效
1. pistol.wav - 手枪射击音效
2. shotgun.wav - 霰弹枪射击音效
3. machine_gun.wav - 机枪射击音效
4. rocket_launcher.wav - 火箭筒射击音效
5. reload.wav - 武器装弹音效

## 车辆音效
1. car_engine.wav - 汽车引擎音效
2. car_horn.wav - 汽车喇叭音效
3. car_crash.wav - 汽车碰撞音效
4. car_skid.wav - 汽车急刹车音效

## 角色音效
1. footstep.wav - 脚步声
2. hurt.wav - 受伤音效
3. death.wav - 死亡音效

## 环境音效
1. ambient_city.wav - 城市环境音效
2. explosion.wav - 爆炸音效

## 界面音效
1. menu_select.wav - 菜单选择音效
2. menu_confirm.wav - 菜单确认音效

你可以使用任何开源免费的音效，例如从Freesound.org或其他开源音效网站下载。
